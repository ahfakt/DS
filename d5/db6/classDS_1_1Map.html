Detailed Description

template<typename K, typename V, typename C = std::less<>, typename ... Cs>
class DS::Map< K, V, C, Cs >

Balanced search tree implementation.

Template Parameters
KKey type of the mapped type to be stored in Map
VValue type to be mapped by K in Map

K can map any V and V-based objects in a Map object, even if V is an abstract class.
