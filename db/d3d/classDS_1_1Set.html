template<typename K, typename C = std::less<>, typename ... Cs>
class DS::Set< K, C, Cs >

Balanced search tree implementation.

Template Parameters
KKey type to be stored in Set

K and any K-based objects can be stored in a Set object simultaneously, even if K is an abstract class.
